Layer 4 - Transport Layer

TCP
- Transmission Control Protocol
- Connection Oriented 
- High reliability
- Websites [[http]](80), [[https]](443), [[docs/Pentesting Notes/Attacks/Hacking Tools/Network Exploitation/Network Services/ftp]], [[ssh]]
- works on 3 way handshake
	- SYN > SYN ACK > ACK
	- SYN - I want to connect on port 443
	- SYN ACK - If port is open and available for connection
	- ACK - when you want to establish connection

UDP
- User Datagram Protocol
- Connectionless 
- Streaming, [[DNS]], VOIP
